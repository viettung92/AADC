This folder contains the SCM Structure for the open challenge (Light Communication between two cars).
There are two different files for two cars. SCM_Structure_KUER_NCT_Car.xml contains a timeout (line 2737) for communication start.

Car 1:
copy SCM_Structure_KUER_LIKE_Car.xml to /Kuer/SCM_Structure/SCM_Structure_CURRENT_KUER.xml

Car 2:
copy SCM_Structure_KUER_NCT_Car.xml to /Kuer/SCM_Structure/SCM_Structure_CURRENT_KUER.xml
